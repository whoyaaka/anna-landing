# Anna landing page

Mobile-first static link-in-bio website for Anna. It contains a main profile page and a separate exclusive page.

## Preview locally

Open `index.html` in a browser. No build step or server is required.

## Deploy

Upload the entire folder to Netlify, Vercel, or any static web host. All paths are relative, so the site also works on GitHub Pages.

## Pages

- `index.html` — main link-in-bio page
- `exclusive.html` — separate page with the Stacked destination

## Update links

Edit the corresponding `href` values in `index.html` and `exclusive.html`.
